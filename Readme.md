# Repositorio: Tareas — Matemática Aplicada 2 (Maestría en Matemática Aplicada)

Este repositorio contiene el código, cuadernos, datos y recursos utilizados para las **tareas (homeworks/assignments)** de la asignatura *Matemática Aplicada 2* de la Maestría en Matemática Aplicada. Está pensado para mantener todo el material organizado, reproducible y fácil de revisar por estudiantes y docentes.


Todo el contenido esta alojada en el notebook __Main.ipynb__, si en algun momento se requiere tener un script por aparte se realizara con la extension __.py__, __.c__ o __.cpp__ .
